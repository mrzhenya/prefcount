Package net.curre.prefcount.gui.theme.skin Description

Contains Look-and-Feel theme/skin classes that contain configurable theme data for all available to PrefCOunt Look-and-Fell themes/skins. Note that we are using the Substance LAF library, so most skins extend the Substance's classes in the org.jvnet.substance.skin package. To find out more about the Substance LAF library, please see the Substance web site.


Substance site: substance.dev.java.net
