PrefCount is an application that assists with counting 
up the results of a game of Preferance. If you don't play 
Preferance this application will do no good for you. 
By the way, Preferance is an old card game that originated 
in Russia in the middle of the XIX century and is still 
widely played across Central Europe and Russia.

INSTALLATION
=======================================================
To install application, simply drag the PrefCount icon 
(the .exe file) to the desired destination (i.e. your 
Desktop).


RUNNING
=======================================================
Navigate to the installation directory and double-click 
on the PrefCount icon (the .exe file).


UNINSTALLING
=======================================================
Navigate to the installation directory and drag the 
PrefCount icon (the .exe file) to Trash.


UPGRADING
=======================================================
Uninstall the previous installation first and then perform 
the new installation.
